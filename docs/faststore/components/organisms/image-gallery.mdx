---

## Import

Import the component from [@faststore/ui](/../docs/faststore/components-index)

```tsx
import { ImageGallery, ImageGallerySelector, ImageZoom } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/organisms/ImageGallery/styles.scss'
```

## Usage

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <ImageGalleryUsage images={imagesUsage} />
    </OverviewSection>
  </Tab>
  <Tab>
    ```tsx
    const ImageComponent = ({ url, alternateName }) => (
      <img data-fs-image src={url} alt={alternateName} />
    )

    const [selectedImageIdx, setSelectedImageIdx] = useState(0)
    const currentImage = imagesUsage[selectedImageIdx]

    <ImageGallery
      images={imagesUsage}
      ImageComponent={ImageComponent}
      selectedImageIdx={selectedImageIdx}
      setSelectedImageIdx={setSelectedImageIdx}
    >
      <ImageZoom>
        <img
          data-fs-image
          src={currentImage.url}
          alt={currentImage.alternateName}
        />
      </ImageZoom>
    </ImageGallery>

    ```

  </Tab>
</Tabs>

## Customization

For further customization, you can use the following data attributes:

`data-fs-image-gallery`

`data-fs-image-gallery="with-selector | without-selector"`

`data-fs-image-gallery-selector`

`data-fs-image-gallery-selector-control`

`data-fs-image-gallery-selector-control-button`

`data-fs-image-gallery-selector-elements`

`data-fs-image-gallery-selector-thumbnail`
